Submit New Café Details

Use this form to submit details for a newly signed café. Nothing goes live automatically — every submission lands in a review queue and Tarryn or Franlo will approve it before it appears on the BOS Dashboard.

Save your store info sheet as CSV directly — one field per row, label in one cell and its value in the next, exactly like the spreadsheet template (Store Name / Walmer Park, then Store Type / Full Store, and so on). One file can hold one café, or several stacked one after another (each starting its own "Store Name" row).

Row labels used: Store Name, Store Type, Location, Seat No., Anticipated Turnover, Store Address, Store Opening Date, Franchisee Invoicing Details, then Representative / FSM / Project Manager / Landlord Contact — each followed by its own Email and Contact No. rows underneath.

Only Store Name and Store Opening Date are required — everything else can be filled in later. Dates can be written as 2026-08-31, 31-Aug-26, or Thursday, July 30, 2026.
Store Type should be one of: Full Store, XS, Kiosk, TEMSA. Region isn't collected here — Tarryn/Franlo add it during review.
