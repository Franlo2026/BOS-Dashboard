Submit New Café Details

Use this form to submit details for a newly signed café. Nothing goes live automatically — every submission lands in a review queue and Tarryn or Franlo will approve it before it appears on the BOS Dashboard.

Expected columns (case-insensitive, any order): Store Name, Region, Type, FSM, Store Address, Store Description, Location, Store Size (m2), Seat No, Anticipated Turnover, Beneficial Occupation Date, Handover Date, Store Opening Date, Representative, Contact Number, Email, Invoicing Details, Landlord, Tenant Coordinator, PM, Notes.

Only Store Name and Store Opening Date are required — everything else can be filled in later. Dates can be written as 2026-08-31 or 31-Aug-26.
Region should be one of: GP, WC, EC, KZN, MP, MPL, LP, NW, NC. Type should be one of: Full Store, XS, Kiosk, TEMSA.
